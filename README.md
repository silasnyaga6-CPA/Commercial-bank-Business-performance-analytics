# 🏦 FinTech Business Performance & Reliability Hub

## 🎯 Executive Summary
This project is an end-to-end analytical ecosystem for a cross-border banking platform (Equity Bank Kenya context). It automates the transition from raw transaction logs to executive-ready insights.

## 🛠️ The Technical Stack
- **SQL (PostgreSQL):** Production-grade P&L, RAG Logic, and WoW Failure Tracking.
- **Python:** Z-Score Anomaly Detection, A/B Testing (scipy), and Linear Regression Forecasting.
- **Power BI:** Interactive 30-measure DAX model for YTD/YoY performance tracking.
- **Advanced Excel:** Dynamic array matrices (LAMBDA/MAKEARRAY) for corridor revenue analysis.

## 🚀 Key Insights from the Data
- **Anomaly Detection:** Identified a $2.5M "Revenue at Risk" pool in customer segments.
- **A/B Testing:** Proved no statistical difference (p=0.95) between USSD and App success rates, pinpointing backend issues over UI.
- **Forecasting:** 95% Confidence Interval modeling predicts a 12% growth trend for EazzyPay.

## 📁 Repository Structure
- `/sql`: Data cleaning and KPI logic.
- `/python`: Statistical modeling and anomaly detection scripts.
- `/excel`: Rapid prototyping and revenue matrices.
- `/power_bi`: DAX measure documentation and dashboard templates.
